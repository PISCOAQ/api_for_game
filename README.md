# API for Game

API Node.js per il recupero dei dati dal gioco.


## Requisiti
- Node.js
- npm
- MongoDB


## Installazione

- Clonare il repository
- Installare le dipendenze
- Per motivi di sicurezza le variabili d'ambiente non sono presenti nel repository, quindi bisogna
  rinominare il file .env.esempio in .env ed inserire al suo interno la propria stringa MongoDb
